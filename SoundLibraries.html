Sound Libraries

Sound Libraries are online repositories of audio files designed to be used by producers. It can be fun to make your own sounds but sometimes its not realistic to create hundreds or possibly thousands of songs for a single project.

Luckily there are hundreds of royalty-free sound libraries availible that are compatible with many DAWS. These sometimes have different MIDI instruments, sound effects, drum kits and more.
These sound libraries are huge time savers and even if you don't like how a sound fits entirely you can just manipulate the effect until its to your liking.

Here's a list of sound libraries I find useful:

While these aren't all the sound libraries availible, you can definitely find most of what you want here.

To install and use a sound library:
  • Click Options -> Preferences
  • Find plug-ins, click on VST
  • Direct REAPER to the folder containing the VST library
  • Click the re-scan button and REAPER will update with the new VST
  • Click insert on the menu bar and select "Insert Virtual Instrument" on New Track or click the FX button on the desired track
  • Select the VST or instrument from the list

Effects

Effects(FX) are a way to change or manipulate audio. There are hundreds of possible effects but the most common ones are:EQ(equalization), compression, reverb, delay and noise reduction.

Virtual instruments like the ones we get from sound libraries, are also considered effects.



The easiest way to apply effects to a track is to click on the "FX" button.
Once you click the button, you'll see your plug-ins and effects.



Adding effects

You normally add effects to the whole track, however it is possible to add effects specifically to items.
Adding an effect through the FX button applies said effect to the whole track. To add an effect:

  • Click the FX button to open list of availible effects
  • Choose the category of effect(VST, instruments etc.) or just select "All Effects"
  • Select your desired effect and click OK
  • That effect is now applied to the track
  • This can be edited at any time in the FX menu


Note: Adding effects will not change or edit the audio file in any way. This is because REAPER uses "non-destructive" editing. This means that you can turn the effect on or off whenever you like. This can be done by clicking the switch on the FX button. It will "turn-off" and this lets you know that there are no effects on the audio. This helps you compare the audio before and after effects.


Printing and Rendering Audio

As I mentioned above, adding an effect will not change the actual audio file at all. The way you apply the effect permanently (called "printing" the effect), you need to render the audo.

To Render a Master Mix of a track, do this:

  • Click on File -> Render
  • Source should be "Master Mix"
  • Select an option in the Bounds mean to select the length of the file
  • Choose a directory to save the file to and name the file
  • Select audio format(wav, mp3 etc.)
  • Click Render

To Render individual tracks:
  • Select the track(s) with Ctrl+Click
  • Click on File -> Render
  • Source should be "Stems Selected Tracks"
  • Select an option in the Bounds mean to select the length of the file
  • Choose a directory to save the file to and name the file
  • Select audio format(wav, mp3 etc.)
  • Click Render

Difference of File Types

WAV Vs. MP3
WAV files tend to be larger as they retain the original audio quality

MP3 files tend to be smaller as they compress the audio to take up less space

Tips:

  • Save your REAPER files as WAVs if you plan on editing the audio more, once you are done you can save the file as an MP3(You won't hear the compression if its done once.)
  • .RPP files are only compatible with REAPER software
  • Everytime you record a take, you'll be prompted to save each track as it's own WAV file. Unless you edit them you won't need to render them seperately. You'll still have to do a render as a Master Mix if you want to combine the tracks.
